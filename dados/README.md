# Análise de Atendimentos - Shineray

Projeto de análise de dados de 500 atendimentos com foco no tempo de resposta ao cliente.

## Objetivo
Avaliar o desempenho dos atendimentos com base no tempo de resposta.

## Classificação
- Bom: até 5 minutos
- Regular: de 6 a 15 minutos
- Ruim: acima de 15 minutos

## Tecnologias
- Python
- Pandas
- OpenPyXL

## Como executar

Instalar dependências:

```bash
pip install -r requirements.txt
